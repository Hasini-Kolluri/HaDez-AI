# 🛡️ HaDez-AI

An AI-powered assistant designed to help Indian citizens report cybercrime incidents through natural conversations. The system understands user complaints, identifies the type of scam, extracts important information, and generates a structured complaint report that can assist law enforcement.

> **Project Status:** 🚧 Under Development

---

## 📌 Project Overview

Reporting cybercrime can often be confusing and overwhelming for victims. This project aims to simplify the reporting process by providing an intelligent chatbot that guides users through their complaint, analyzes the incident, and generates a well-structured report.

The long-term vision is to support multilingual conversations, voice input, evidence uploads, and a police analytics dashboard for identifying fraud patterns across multiple complaints.

---

## 🎯 Objectives

* Assist users in reporting cybercrime incidents through conversational AI.
* Detect and classify different types of cyber scams.
* Extract important entities such as phone numbers, UPI IDs, URLs, bank accounts, and transaction amounts.
* Generate police-ready complaint summaries.
* Provide immediate guidance and recommended next steps for victims.
